# Simple Interest Calculator

This project calculates simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Inputs:
- Principal amount
- Rate of interest
- Time period

## Output:
- Simple Interest

## Example:
Principal = 1000  
Rate = 5%  
Time = 2 years  

Simple Interest = (1000 × 5 × 2) / 100 = 100
